
# Налачо работы с проектом #

### Установка компонентов bower ###

```javascript
bower install --save
bower install --save #name-plugin#
```
### Установка компонентов gulp ###

```javascript
npm install --save-dev
npm install --save-dev #name-plugin#
```
### Старт работы ###

```javascript
gulp
```
### Сборка проекта ###

> обычный
```javascript
gulp build
```
>минифицированый
```javascript
gulp build-min
```
